This is an online shopping application where multiple users can search for products, view their details, purchase single products or add them 
to the cart to purchase a group of products. the users can also view their placed orders history

The application is python Django based.  HTML, Javascript, Bootstrap (all integrated in django) have also been used for the front end

The different components of the product (e.g folders carts, search, products etc) have been added as a separate application to the same django project and can be handled and modified 
independently

The src folder contains all the developed modules for this project
